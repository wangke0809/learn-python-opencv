# 为什么要学习Pyhton-OpenCV?

- 使用C/C++配置太麻烦（但是文档齐全）  
- 快速验证想法  
- 便于与深度学习结合  
- 便于实现各种玩具级小项目  
- 性能若有要求，便于转向C/C++实现  

## 学习思路

主要参考官网文档：https://docs.opencv.org/3.4.2/

目前最新稳定版为`3.4.2`，没有必要纠结学哪个版本，基本函数出入参没什么变化，有变化的具体查一下手册即可。

## 运行环境

Python3.5+  
opencv-python3.4.2+  
jupyter notebook
